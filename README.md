# Roomify - 旅館空房管理與預訂平台

這是一個專為小型旅館 / 民宿打造的現代化空房查詢與管理平台。前端採用原生 **HTML5 / CSS3 / JavaScript (ES6)** 開發，視覺走**色彩明亮簡潔、字體圓潤**風格。

---

##  專案檔案結構 (Project Structure)

├── index.html      # 網頁結構 (HTML)
├── style.css       # 視覺樣式與配色 (CSS)
├── script.js       # 互動邏輯與狀態控制 (JavaScript)
└── README.md       # 專案說明文件與未來 API 規格
